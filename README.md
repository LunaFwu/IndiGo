# IndiGo

A modern custom UI / wrapper for the Hydrogen Roblox executor on macOS.

## Requirements

- Hydrogen installed and working

## Features

- Clean minimal UI
- Script load, edit, execute
- More

## Usage

2. Run the UI wrapper.
3. Load or paste your script.
4. Click "Execute".

## Notes

- Hydrogen must be installed before using this UI.
- This UI does **not** include Hydrogen itself.
